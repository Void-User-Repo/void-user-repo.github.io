** Installation

*** Add the repository configuration
#+begin_src shell
echo "repository=https://github.com/Void-User-Repo/void-user-repo.github.io/releases/download/latest" | sudo tee /etc/xbps.d/10-zen.conf
#+end_src

*** Sync and install
#+begin_src shell
sudo xbps-install -S linux-zen linux-zen-headers
#+end_src
